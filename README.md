# openai-stream-parser

本项目主要提供前端如何解析 OpenAI Stream 的方案。

且附加 DEMO 说明如何在不同的环境下如何使用解析器，

## 功能特性

- [✅] 浏览器环境解析 OpenAI Stream（UMD）
- [✅] Node 环境解析 OpenAI Stream （CJS）
- [✅] ES6 环境解析 OpenAI Stream （ESM）
- [✅] 浏览器环境 DEMO 示例
- [❌] Node 环境 DEMO 示例
- [❌] ES6 环境 DEMO 示例
